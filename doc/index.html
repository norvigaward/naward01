Analyzing Web Domain Vulnerability

“Time-based analytics of the evolution of security risks of used web stacks on the web. Security on the web is a topic which gained a lot of traction recently. Through the proposed analysis we can gain an insight of the measures taken by the web community by looking at the web stacks used.”

by Sietse T. Au and Wing Lung Ngai

This project "Analyzing Web Domain Vulnerability - Norvig Award 2014"

is in partial fulfillment of the requirements for the course IN4144 Data Science, at Delft University of Technology.
